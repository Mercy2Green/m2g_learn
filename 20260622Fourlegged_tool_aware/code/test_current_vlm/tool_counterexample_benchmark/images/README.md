# Image Naming Rules

Put images under the matching task folder:

```text
images/task_001_multi_bottles_visible_container/
```

Use sequential names:

```text
image_01.jpg
image_02.jpg
image_03.jpg
```

Supported extensions: `.jpg`, `.jpeg`, `.png`, `.webp`, `.bmp`.

Each image should be:

- Robot first-person view or close to it.
- A real or realistic household, office, kitchen, bedroom, lab, or hallway scene.
- Free of text labels that reveal the answer.
- Natural rather than puzzle-like.
- Clear enough to identify task-relevant objects and possible helpers.

For "no visible helper" tasks, do not include obvious trays, bags, boxes, baskets, rods, or cleaning tools in the current view. The task folder README may mention that nearby search could be reasonable.
