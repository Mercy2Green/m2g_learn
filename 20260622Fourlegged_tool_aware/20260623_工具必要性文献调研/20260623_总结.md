# Weekend Executive Summary: AIGC+Sim × Mobile × Tool-Aware

## One-Sentence Project Focus

> 研究如何组合 2025–2026 的 AIGC/simulation 数据方法、mobile/quadruped manipulation 表示与 tool-aware 方法，使四足臂根据工具状态输出 `direct / use_tool / reposition / abort-recovery`。

## Intersection Audit Conclusion

- 找到一篇明确 L3：GET-USE（simulation + bimanual mobile manipulation + generalized tool use）。
- 找到 L2C：SAGA（quadruped mobile manipulation + structured affordance）。
- 找到 L2A：SLIM、MobileManiBench、SG-VLA、MoManipVLA、RAKOMO。
- 找到 L2B：RIGVid、Counterfactual Tool Use、Robustness-Aware Selection、iTuP、RobotSmith。
- 仍未找到可运行、覆盖四足臂 + AIGC/sim pairs + tool necessity + 四类 mode 的 baseline。

> 本周汇报不是 mobile manipulation survey，也不是 tool-use survey，也不是 AIGC robot learning survey；它是三者交集的 related work / baseline / limitation survey。

## What 2025–2026 Works Provide

1. **AIGC/sim:** generated videos、future/action models、simulation generators。
2. **Mobile/quadruped:** base-arm reachability、reposition、whole-body state/action interfaces。
3. **Tool-aware:** affordance、tool selection、suitability、physics-conditioned use。
4. **Runnable components:** Qwen3-VL、RoboTwin 2.0、Aff-Grasp、GR00T demo。

## What No Current Work Provides

- public quadruped-arm tool-necessity pair generator；
- native four-mode output；
- verified AIGC tool-state labels；
- mobile VLA tool-necessity benchmark；
- fixed-base tool method with base reposition；
- runnable L3 baseline。

## What to Tell the Advisor

1. L3 evidence exists but is sparse and non-runnable。
2. Our motivation is closing data + embodiment + tool-state decision, not inventing tool use。
3. First-stage evaluation should compose existing baselines, not invent a new backbone。
4. Simulation owns labels；AIGC only extends visuals after verification。
5. Mobile/quadruped state must include base/tool relative reachability and reposition feasibility。

## What Not to Claim

- GET-USE solves our quadruped necessity problem。
- SAGA affordance equals tool necessity。
- AIGC video provides reliable actions or labels。
- mobile papers are tool-aware evidence。
- fixed-base tool papers cover mobile reposition。
- Qwen3-VL/GR00T are intersection literature；they are baselines。
