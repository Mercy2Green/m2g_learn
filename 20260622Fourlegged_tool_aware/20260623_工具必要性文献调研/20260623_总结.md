# 2025–2026 Executive Summary

## One-Sentence Project Focus

> 基于 2025–2026 的 AIGC、simulation、VLM/VLA 和移动操作项目，研究四足臂机器人能否根据工具可用性、可达性、合适性和 direct feasibility，在 `direct / use tool / reposition / abort-recovery` 间做出正确决策。

## What 2025–2026 Works Already Provide

- Qwen3-VL：可直接运行的视觉推理 baseline。
- π0.5/openpi、GR00T N1/current N1.7：开放 robot foundation policy。
- Gemini Robotics/ER：强但 restricted 的 capability reference。
- Video Policy、UWM、DreamGen、Cosmos：生成视频、未来预测和 pseudo-action pipeline。
- RoboTwin 2.0：开放 simulation data generator 和 100k+ trajectories。
- Human2LocoMan：开放四足 manipulation 数据、代码和模型。
- Aff-Grasp：开放 affordance 数据、模型和代码。
- 2025–2026 tool papers：tool property、grasp physics 和 robustness 会改变工具使用。

## What They Do Not Provide

- 没有公开 baseline 原生输出本项目四类 mode。
- 没有公开 generator 直接产生 tool/no-tool、usable/unusable、reachable/unreachable matched pairs。
- AIGC 视频不默认保持工具状态、物理结果或动作标签。
- mobile/quadruped work不证明 tool necessity。
- 工具论文主要解决 which/how，而非系统比较 direct versus tool use。

## Baseline Status

| Level | Baselines |
|---|---|
| Runnable / near-runnable | Qwen3-VL、RoboTwin 2.0、Aff-Grasp、GR00T N1.7 demo |
| Adaptable later | π0.5/openpi、GR00T finetuning、UWM、Video Policy、DreamGen、Human2LocoMan |
| Conceptual only | iTuP、robustness-aware tool selection、counterfactual tool use、Guardian、Dream2Fix、MoManipVLA、SG-VLA |
| Restricted | Gemini Robotics / Robotics-ER |

## Strongest Motivation

> 近两年工作已经提供成熟 backbone、生成模型和仿真数据平台，但其数据与评测没有显式隔离工具必要性。最小研究问题是：基于现有 2025–2026 baseline，仿真生成的 matched tool-state pairs 是否改善四类 mode decision；AIGC 是否只有在 tool-state/label verification 后才有辅助价值。

## What Not to Claim

- 第一阶段不建议训练新的通用 VLM，因为这不利于隔离 tool-aware 机制，且已有 2025–2026 foundation models 更适合作 baseline。
- 不声称 AIGC 直接提供可靠 action supervision。
- 不声称 world model 已学会意图或 tool necessity。
- 不声称四足平台本身是 novelty。
- 不把 custom classifier 称为 existing strong baseline。
- 不把 conceptual/no-code paper 写成可复现实验。
