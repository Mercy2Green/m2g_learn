# Weekend PPT: AIGC+Sim Data for Tool-Aware Quadruped-Arm Mobile Manipulation

## Slide 1 — Project Focus

**Key message:** 汇报研究三元交集，不是三个独立 survey。

- AIGC/simulation data；
- quadruped-arm/mobile manipulation；
- tool-aware mode decision；
- 输出：direct/use_tool/reposition/abort。

**Supporting works:** GET-USE, SLIM, RIGVid。  
**Intersection level:** L3/L2 adjacency。  
**Suggested figure:** 三圆 Venn diagram，中心写 four-mode decision。

## Slide 2 — What Tool-Aware Means Here

**Key message:** 工具功能、工具状态和移动可执行性必须联合决定 mode。

- direct feasibility；
- tool availability/suitability；
- tool reachability；
- base reposition feasibility；
- failure/abort condition。

**Supporting works:** iTuP, Robustness-Aware Selection, RAKOMO。  
**Intersection level:** L2B + L2A。  
**Suggested figure:** 状态变量→四类 mode 真值图。

## Slide 3 — Three Required Ingredients

**Key message:** 任意单一方向都不足。

- data generation 提供 paired states；
- mobile embodiment 提供 base-arm constraints；
- tool-aware methods提供 suitability/affordance；
- foundation baseline提供推理/动作载体。

**Supporting works:** RoboTwin 2.0, Human2LocoMan, Counterfactual Tool Use, Qwen3-VL。  
**Intersection level:** L1/L2 components。  
**Suggested figure:** ingredient-to-system pipeline。
